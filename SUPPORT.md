# Support

Use GitHub Issues for reproducible bugs, Marketplace or packaged VSIX problems, and asset import or asset rights questions:

https://github.com/Hitsuki-Ban/guruguru-codechan/issues

Before filing an issue, use the structured issue forms:

- Bug reports: `.github/ISSUE_TEMPLATE/bug_report.yml`
- Asset rights and import reports: `.github/ISSUE_TEMPLATE/asset_rights.yml`

Please include the extension version, install source, VS Code version, operating system, reproduction steps, and screenshots or logs when they are relevant.

Bundled sample assets are non-commercial. User-imported assets remain governed by the rights of the user or the user's licensors. Do not attach proprietary character assets unless you have the right to share them in a public issue.

Feature ideas are welcome, but the `0.x` line prioritizes release blockers, VS Code integration quality, performance, asset safety, and packaging reliability.
